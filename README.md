# Weather Derivatives Project

---

## Purpose of the Project

The goal of this project is to build a pricing model for weather derivatives

---

## Project Structure

### **`meteostat_data_collection.py`**
Collect historical weather data from the Meteostat API.  

### **`descriptive_stats.ipynb`**
Jupyter notebook for:
- Checking missing values  
- Descriptive statistics  
- Data visualization  

### **`functions.py`**
Contains utility functions for:
- Data processing  
- Statistical Testing 
- Weather index calculation (HDD, CDD, CAT)

### **`functions1.py`**
Includes functions for:
- Model estimation  
- Simulation of stochastic processes    

### **`model_fit_`**
Module or folder used to:
- Fit models to each dataset  
- Store estimated parameters    

### **`contract_pricing`**
Implements pricing method for weather derivative contracts
